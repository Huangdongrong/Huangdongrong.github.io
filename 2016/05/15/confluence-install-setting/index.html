什么是Confluence

Confluence是一个企业知识库管理软件，用于构建企业wiki，通过它可以实现团队成员之间的协作和知识共享，并且还可以集成缺陷(Bug)跟踪管理工具 - JIRA，
还可以集成更多插件工具等等，相关插件请查阅Atlassian官网文档。

Confluence安装安装前准备

Confluence下载：https://www.atlassian.com/software/confluence/download
数据库MySQL下载：http://dev.mysql.com/downloads/mysql/
汉化包：Confluence-5.9.4-language-pack-zh_CN.jar
MySQL驱动包：mysql-connector-java-5.1.35.jar
我的github下有所需要的包：https://github.com/Huangdongrong/soft-and-jar-repository/blob/master/confluenceInstall.rar

Confluence安装

  1. 打开atlassian-confluence-5.9.7-x64.exe，下一步
  2. 选择自定义安装Confluence
  3. 选择Confluence安装目录
  4. 选择Confluence数据保存目录
  5. 更改端口，以免端口冲突

    6.下一步下一步直到安装完成，打开浏览器http://localhost:8090/进入设置页面，证明安装成功，接下来看下面Confluence配置

Confluence配置

  1. 进入安装目录下xx\Atlassian\Confluence点击stop_service.bat停用服务
  2. 把下载好的MySQL驱动包mysql-connector-java-5.1.35.jar，放到安装目录xx\Program Files\Atlassian\Confluence\confluence\WEB-INF\lib下
  3. 进入安装目录下xx\Atlassian\Confluence点击start_service.bat再启动服务，重新打开浏览器http://localhost:8090/进入设置页面
  4. 第一个选项是试用版的，我们选择第二个产品，点击Next Next
  5. 把日历与问题总汇插件都选择，点击Next Next
  6. 进入到授权页面，点击输入框下面链接，进入到atlassian登录页，没有帐号请进行注册atlassian帐号
  7. 登录后，回到我们的授权页面，点击第一个输入框下面的链接，跳转到填写licence页面
  8. 填写licence信息，点击Generate Licence,跳转到已经注册好的Licence列表页
    Product:注册的产品，选择Confluence（Server）
    Licence:自己随便命名
    Server ID:授权页面的Server ID
  9. 选择刚才生成Licence，展开内容，复制Licence Key到授权页面的第一个输入框
  10. 授权页面第二个输入框的Licence Key也一样生成，点击下面的链接，跳转到填写licence页面
  11. 填写licence信息，点击Generate Licence,跳转到Licence详细页
    Product:注册的产品，选择Team Calendars for Confluence（Server）
    Licence:自己随便命名
  12. 复制Licence Key到授权页面的第二个输入框
  13. 授权页面第三个输入框的Licence Key也一样生成，点击下面的链接，跳转到填写licence页面
  14. 填写licence信息，点击Generate Licence,跳转到Licence详细页
    Product:注册的产品，选择Questions for Confluence（Server）
    Licence:自己随便命名
  15. 复制Licence Key到授权页面的第三个输入框
  16. Licence Key都填好，Next，进到选择数据库，使用扩展数据库
  17. 点击External Database,如果配置前没有把MySQL驱动包mysql-connector-java-5.1.35.jar，放到放到安装目录xx\Program Files\Atlassian\Confluence\confluence\WEB-INF\lib下，会有以下提示缺少驱动包，先停服务，再启动，再刷浏览器如果出错
    恭喜你又重装了（请查看卸载篇），成功继续点击External Database

  18. 打开数据库安装目录下的xx\MySQL\MySQL Server 5.1\my.ini文件

    1. default-character-set设置为utf-8

    2. 在最后面增加以下设置编码

  20. 设置数据库信息

  21. 创建一个例子wiki，能让我们感受到人家怎么在里面写文章的 ，Next
  22. 设置管理员Confluence帐号信息，Next
  23. 以下都是设置管理员信息，都可以跳过，点击SKip SKip SKip SKip
  24. 为刚才创建的例子，创建一个存储空间

  26. 恭喜你，到这里就配置完成了，打开http://localhost:8090/#all-updates，可以看到空我们创建的Demo空间

Confluence汉化

进入安装目录下xx\Atlassian\Confluence点击stop_service.bat停用服务，把下载好语言包Confluence-5.9.4-language-pack-zh_CN.jar放到xx\Atlassian\Confluence\confluence\WEB-INF\lib 目录下，
xx\Atlassian\Confluence点击start_service.bat再启动服务，重新打开浏览器http://localhost:8090/ 汉化成功。
