Confluence 使用(二)

  • 运维

Confluence写文章

首先我们需要了解什么是Confluence空间，空间可以理解我们计算机的硬盘，有硬盘才可以装东西，所以文章前我们需要创建空间。

  1. 点击创建空间，选择创建空白空间，下一步
  2. 输入空间名标识空间
  3. 删除系统自动生成的首页
  4. 创建文章，点击创建页面，编写文章，标题是必填的

如果创建到某个分类下，选择文章节点，如下选择“1.confluence使用”，点击创建，编写文章完成后，效果如下：

Confluence修改文章

在页面树结构下，选择节点文章，点击编辑。

Confluence删除文章

在页面树结构下，选择节点文章，点击更多，展开操作，点击删除。

Confluence备份数据

  1. 点击一般配置，管理员帐号登录
  2. 每日备份管理，默认系统开启每天自动备份，备份文件会保存到xx\Atlassian\Application Data\Confluence\backups 目录下

  3. 手动备份，点击备份与还原菜单，点击备份，备份文件会保存到xx\Atlassian\Application Data\Confluence\temp 目录下

    Confluence还原恢复数据

    1. 打开数据库安装目录下的xx\MySQL\MySQL Server 5.1\my.ini文件

      • default-character-set设置为utf-8

      • 在最后面增加以下设置编码

    2. 点击备份与还原菜单，还原方式有两种，使用自动备份文件或手动备份文件都可以还原恢复当前备份时间的数据。

      Confluence添加用户

      点击用户菜单，添加用户。

      Confluence分配权限

  4. 点击用户安全菜单，编辑
  5. 一般会设置匿名可以访问
